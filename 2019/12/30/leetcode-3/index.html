LeetCode-34 在排序数组中查找第一个和最后一个位置

发表于 2019-12-30 分类于 学习笔记 Valine：

依然是二分法思想

给定一个按照升序排列的整数数组 nums，和一个目标值 target。找出给定目标值在数组中的开始位置和结束位置。

你的算法时间复杂度必须是 O(log n) 级别。

如果数组中不存在目标值，返回 [-1, -1]。

分析：通过二分法查找，在命中一个target后，顺序向前和向后扫描target
